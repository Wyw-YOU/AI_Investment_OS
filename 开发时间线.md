## 风险登记表（Top 5）

| # | 风险 | 概率 | 影响 | 缓解方案 |
|---|------|:----:|:----:|---------|
| R1 | LLM prompt 调试周期超预期 | 高 | 高 | 预设测试用例集；先用小模型快速迭代 |
| R2 | Agent 输出格式不稳定 | 中 | 高 | JSON Schema 校验 + 自动重试 + fallback |
| R3 | 行情数据 API 限流/不可用 | 中 | 中 | 多数据源备用（akshare + tushare + eastmoney） |
| R4 | 前端 Stock Workspace 复杂度溢出 | 中 | 中 | 使用 ECharts/lightweight-charts 降低图表开发成本 |
| R5 | Token 成本超预算 | 低 | 中 | 开发用 GPT-4o-mini；生产用 prompt caching |
