# AI Investment OS - 开发进度报告

**项目名称**: AI Investment OS (Multi-Agent Investment Research Operating System)
**报告日期**: 2026-07-06
**当前版本**: V1.0 MVP

---

## 项目概要

| 项目 | 详情 |
|------|------|
| 技术栈 | FastAPI + LangGraph + Next.js 14 + SQLite + Redis |
| 部署环境 | 2C4G Ubuntu + Docker Compose |
| LLM 接入 | DeepSeek / Xiaomi MiMo (OpenAI 兼容 API) |
| Agent 数量 | 8 个专业 Agent |
| 工作流模式 | LangGraph 并行 Fan-Out/Fan-In |

## 关键技术决策

| 决策 | 选择 | 替代方案 | 原因 |
|------|------|----------|------|
| 数据库 | SQLite | PostgreSQL | 2C4G 内存限制, 节省 ~300MB |
| 向量库 | 暂不引入 | Qdrant/FAISS | MVP 阶段内存敏感, 后续可扩展 |
| LLM 接入 | OpenAI 兼容 API | 原生 API | 统一接口, 一键切换 DeepSeek/MiMo |
| 工作流 | LangGraph | Celery Chain | 原生并行支持, 状态管理完善 |
| 数据源 | AKShare | Tushare | 免费无需 Token, A 股数据全 |
| K 线图 | ECharts | TradingView Widget | 更灵活的自定义, 更小的体积 |

## 下一步计划

1. **WebSocket 实时推送**: Agent 分析进度实时推送到前端

5. **自动化调度**: 每日定时扫描、报告生成
6. **Strategy 中心**: 自然语言策略创建 + 回测
