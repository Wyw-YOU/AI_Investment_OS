## 2. 快速启动测试

### 2.1 本地启动

```bash
# 终端 1: 启动后端
cd backend
python -m venv .venv && .venv\Scripts\activate
pip install -r requirements.txt -i https://mirrors.tencent.com/pypi/simple/ --trusted-host mirrors.tencent.com
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 终端 2: 启动前端
cd frontend
npm install
npm run dev
```

或双击 `start-local.bat` 一键启动。

### 2.2 Docker 启动

```bash
cp .env.example .env
# 编辑 .env 填入 LLM_API_KEY
docker compose up -d --build
```

### 2.3 验证服务

```bash
# 健康检查
curl http://localhost:8000/health
# 预期: {"status":"ok","version":"1.0.0"}

# 测试管理员登录
curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@aiios.com","password":"admin123456"}'
# 预期: {"code":0,"data":{"user":{"email":"admin@aiios.com",...},"token":"eyJ..."}}
```

## 4. API 端点测试

### 4.1 公开端点 (无需登录)

```bash
# 热门股票
curl http://localhost:8000/api/stocks/hot

# 个股详情
curl http://localhost:8000/api/stocks/600519

# K 线数据
curl http://localhost:8000/api/stocks/600519/history

# 技术指标
curl http://localhost:8000/api/stocks/600519/indicators

# 市场概览
curl http://localhost:8000/api/market/overview
```

### 4.2 需认证端点

```bash
# 先获取 token
TOKEN=$(curl -s -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@aiios.com","password":"admin123456"}' \
  | python -c "import sys,json; print(json.load(sys.stdin)['data']['token'])")

# 触发 AI 分析
curl -X POST http://localhost:8000/api/agent/run \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"stock_code":"600519"}'
# 预期: {"code":0,"data":{"task_id":1,"analysis_id":"1_600519","status":"pending"}}

# 查询分析状态 (用返回的 task_id)
curl http://localhost:8000/api/agent/status/1

# 分析历史
curl -H "Authorization: Bearer $TOKEN" http://localhost:8000/api/agent/history

# 研究空间列表
curl -H "Authorization: Bearer $TOKEN" http://localhost:8000/api/workspace
```

## 5. 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| "Not authenticated" | 未登录 | 用管理员账号登录 |
| 热门股票"暂无数据" | 后端未启动或 AKShare 网络问题 | 确认后端在 8000 端口运行，检查网络 |
| K 线图空白 | 非交易时间数据问题 | 正常，系统会使用缓存数据 |
| AI 分析一直显示"运行中" | LLM API Key 未配置或超时 | 检查 .env 中 LLM_API_KEY |
| WebSocket 连接失败 | 前端无法直连后端 8000 端口 | 确认后端运行，检查防火墙 |
| 管理员账号登录失败 | 首次启动未自动创建 | 重启后端，检查控制台日志 |
