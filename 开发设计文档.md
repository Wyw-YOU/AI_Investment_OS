  # 2. 🧠 核心运行内核（LangGraph Production化）

  ------

  # 2.1 State设计（核心🔥）

  这是你系统“灵魂结构”

  ```
  class InvestmentState:
      user_id: str
  
      stock_pool: list[str]          # 当前分析股票
      current_stock: str
  
      market_data: dict
      financial_data: dict
      news_data: list
      indicators: dict
  
      agent_outputs: dict
      agent_confidence: dict
  
      risk_profile: dict
  
      portfolio: dict
  
      candidate_pool: list[str]
  
      final_report: dict
  
      events: list
  
      decision: str
  ```

  ------

  ## 🧠 关键设计思想

  > ❗所有 Agent 都不直接返回结果
  >  ❗全部写入 State

  👉 这一步是“demo → production”的关键

  ------

  # 3. 🧠 Agent Runtime Engine（关键工程模块）

  ------

  # 3.1 Agent统一接口（必须标准化）

  ```
  class BaseAgent:
      name: str
  
      def run(state: InvestmentState) -> dict:
          """
          输入：统一state
          输出：结构化结果
          """
          pass
  ```

  ------

  ## 📌 强制规范（非常重要）

  每个 Agent 必须返回：

  ```
  {
      "output": {},
      "confidence": 0.0,
      "evidence": [],
      "reasoning": ""
  }
  ```

  ------

  # 3.2 Agent Registry（工程化核心）

  ```
  AGENT_REGISTRY = {
      "planner": PlannerAgent(),
      "finance": FinanceAgent(),
      "technical": TechnicalAgent(),
      "news": NewsAgent(),
      "risk": RiskAgent(),
      "judge": JudgeAgent(),
      "portfolio": PortfolioAgent(),
      "report": ReportAgent()
  }
  ```
