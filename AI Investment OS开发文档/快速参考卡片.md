## 常用命令

### 开发环境
```bash
# 启动所有服务
docker-compose up -d

# 运行后端
cd services/api-gateway
uvicorn main:app --reload

# 运行前端
cd apps/web
npm run dev

# 运行测试
pytest
npm test
```

### Git工作流
```bash
# 创建功能分支
git checkout -b feature/xxx

# 提交代码
git add .
git commit -m "feat: add xxx"

# 推送并创建PR
git push origin feature/xxx
```

### 数据库操作
```bash
# 创建迁移
alembic revision --autogenerate -m "add xxx"

# 执行迁移
alembic upgrade head

# 回滚迁移
alembic downgrade -1
```
