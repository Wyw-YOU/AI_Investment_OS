# AI Investment OS - 开发周期规划文档

> Multi-Agent Investment Research Operating System
> Development Timeline & Sprint Planning

**文档版本**: V1.0
**创建日期**: 2026-07-06
**规划周期**: 20周（10个Sprint）

---

## 1. 项目总览

### 1.1 项目范围

**AI Investment OS** 是一个基于多智能体（Multi-Agent）协作的智能投研系统，包含：
- 8个专业Agent（Planner, News, Financial, Technical, Macro, Risk, Quant, Report）
- 6个核心领域（Identity, Investment, Workspace, Agent, Analytics, Automation）
- 完整的前后端系统
- Docker容器化部署

### 1.2 团队配置

**推荐团队规模**: 3-4名开发者

| 角色 | 人数 | 职责 |
|------|------|------|
| 技术负责人 | 1 | 架构设计、代码审查、技术决策 |
| 后端开发 | 1-2 | API服务、Agent开发、数据库 |
| 前端开发 | 1 | UI组件、交互设计、状态管理 |

### 1.3 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Next.js 14+, React 18, TailwindCSS, ECharts |
| 后端 | FastAPI, Python 3.11+, SQLAlchemy 2.0 |
| AI/ML | LangGraph, LLM APIs, Qdrant |
| 数据库 | PostgreSQL, Redis, MinIO |
| 部署 | Docker Compose, Nginx |
